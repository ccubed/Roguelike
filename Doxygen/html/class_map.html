void Map::DisplayMap ( std::ostream &  output,
int  x,
int  y 
)

First of two DisplayMap functions. This one takes an x,y offset to indicate where the player's new position on the map falls.
That is, if the player moves up one space, this function would be called like: DisplayMap(output,0,-1)
That is: Display the Map and move the player 0 x spaces, and -1 y spaces (this is actually up)

Parameters
[in]outputOutput stream to print the map to
[in]xX Offset to move the player before display
[in]yY Offset to move the player before display
